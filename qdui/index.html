QD UI 文档 v1.0

0. 概述

针对公司将在企点QQ中内嵌页面所做的组件

使用此组件需同时引入【QDUI.css、 QDUI.js】两个文件，后者依赖于 jQuery 2.1.1+

如果你看到了一个 css class 是以 '-' 开头的，表示这是前一个 class 的私有 class，或者叫约束 class，往往用作区别同一组件的不同样式

QDUI.js 将在全局生成一个 QD 对象，实现组件交互，定义了若干事件和方法，将在下文分别提及

下文中所有 $ 打头的变量均指使用 $() 后转换为 jQ Obj 的对象

组件的 class 全部使用 [qd] 为前缀；组件选中状态全部使用 [class="on"]，禁用全部使用 [disabled]

选中的单选 禁用的按钮
选中的单选 禁用的按钮

墙裂建议使用 body 代理各组件事件， [safe-click] 事件可以免去组件的活性判断去看看

安全事件：组件完成 UI 交互后，在【未被禁用】的前提下，会自动触发自定义的 [safe-click] 事件

使用 [safe-click] 事件绑定代理，可以免去组件的活性判断

可以配合 $ 扩展的两个方法 $el.able() $el.disable() 来激活、失活组件：本质上就是删除、添加 [disabled]

另外 $el.isDisabled() 会返回组件是否被禁用

注意：两个控制方法没有以 d 结尾；检测方法是以 d 结尾的

按钮1 按钮2 激活它俩 失活它俩
按钮1 按钮2 激活它俩 失活它俩 
$('body')
    .on('safe-click', '.J_demoSafeClick', function() {
        QD.toast( $(this).text() + ' clicked');
    })
    .on('safe-click', '.J_demoActive', function() {
        $('.J_demoSafeClick').able();
    })
    .on('safe-click', '.J_demoFreeze', function() {
        $('.J_demoSafeClick').disable();
    })

10. QD.toast()

该方法指定了两个参数 QD.toast(text, time)； text 为显示的文本（必填）；time 为显示的时长（毫秒数），默认 2000

2s 5s
QD.toast('2s 后消失'); QD.toast('5s 后消失', 5000);

11. QD.open()

由于 btn 非常规的设计布局，btn 仍以 html 片段形式填充

btn 事件绑定仍通过指定 [class="J_*"] 代理；[class="qd-dialog-cancel"] 可以实现【取消】功能

按钮默认居右，如需放置在左侧，请给 btn 添加 [class="left"]，且在拼接 html 时，将居左的按钮优先拼接

Open 
QD.open({
    title: '标题',
    content: 'HTML 片段',
    btn:
        '新增联系人' +
        '取消' +
        '确定'
});

.on('safe-click', '.J_demoDialogAdd', function() {
    QD.toast('点击了添加联系人');
})
.on('safe-click', '.J_demoDialogComfirm', function() {
    QD.toast('点击了确定');
})

由于 btn 是以 html 形式填充，所以未实现类似 layer.open({ close: function(){..} }) 的回调绑定，只能通过 事件代理

QD.open() 将会返回一个 dialog 的 uuid，可以通过 QD.close(uuid) 来关闭它

在使用 [safe-click] 代理的前提下，QD.closeWith($btn) 也可以关闭按钮所在的 dialog

QD.closeWith($btn) 这种方式虽稍显愚蠢，但根据当前的项目情况，已是最佳之选

弹框 
.on('click', '.J_demoDialogClose', function() {
    QD.open({
        title: '主动关闭',
        content: '关闭',
        btn: '关闭'
    });
})
.on('safe-click', '.J_demoDialogCloseBtn', function() {
    QD.closeWith( $(this) );
})

.on('safe-click', '.J_demoLoading', function() {

})
